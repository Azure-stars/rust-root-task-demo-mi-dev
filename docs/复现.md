# 运行

## 如何运行？

```shell
# 添加磁盘镜像，注意如果出现 -func option not found这类错误，把-func改成-mode即可，因为Linux不同版本支持不一样
bximage -q -hd=16 -func=create -sectsize=512 -imgmode=flat mount.img

# 运行DOCKER并进入
make -C docker rm-container && 
make -C docker/ ARCH=aarch64 run && 
make -C docker/ ARCH=aarch64 exec

# DOCKER中执行
make run
```
